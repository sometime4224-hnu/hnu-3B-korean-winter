추측 정확성 퀴즈 (10문항)
목표: -을 텐데요 / -을 것 같아요 / -을걸요의 근거 유무와 강도(확신) 차이를 상황으로 변별하기
-을 것 같아요: 근거가 약하거나 일반 추정. 강도 중-약.
-을걸요: 화자 확신이 비교적 큼(경험/직감). 강도 중-강.
-을 텐데요: (상대도 알 법한 근거/상황) + “그럴 것”을 전제로 한 대화 유도(제안/지적/아쉬움). 강도 중-강, 맥락 의존.
조작: 보기 클릭 → 즉시 채점/해설. N 다음, P 이전, R 다시 시작

판단 기준(수업용)

  • 근거(증거): 직접 봄/들음/규칙/습관/상황 단서
  • 확신 강도: 낮음(가능성) ↔ 높음(거의 확실)
  • 화용: 상대 반응 유도(제안/지적/아쉬움)면 -을 텐데요가 잘 붙음

표현 빠른 정리

1) -을 것 같아요
· “그럴 가능성이 있어요.” (조심스러운 추정)
· 근거가 약해도 가능. 일상적/중립적.
2) -을걸요
· “아마(거의) 그럴 거예요.” (확신 더 큼)
· 경험/직감/상황판단으로 단정에 가까움(하지만 예언은 아님).
3) -을 텐데요
· “그럴 텐데(그래서…)” 전제 + 다음 행동/평가 유도
· 제안/지적/아쉬움/대조(‘그랬을 텐데요…’)로 자주 씀.
수업 팁: 학생에게 “근거가 약하면 -을 것 같아요, 확신이 크면 -을걸요, 상대 반응을 끌어내면 -을 텐데요”로 1차 분류시키고, 예외(문맥)를 토론하세요.
